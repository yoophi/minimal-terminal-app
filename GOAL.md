# Goal

macOS에서 사용할 수 있는 안정성 우선의 터미널 에뮬레이터를 개발한다.

## Project Scope

- 개발 언어: Rust
- 대상 플랫폼: macOS
- 최우선 가치: 안정성

## MVP Scope

첫 번째 버전은 macOS에서 기본 셸을 안정적으로 실행할 수 있는 단일 창 터미널 에뮬레이터를 목표로 한다.

### 포함할 기능

- macOS 기본 셸 실행
- PTY 기반 프로세스 실행
- 키보드 입력 전달
- 표준 터미널 출력 표시
- ANSI escape sequence 기본 처리
- 고정폭 폰트 렌더링
- 스크롤백 지원
- 텍스트 선택
- 복사 및 붙여넣기
- 창 크기 변경 시 터미널 크기 동기화
- 프로세스 종료 상태 표시
- 비정상 종료 시 앱 크래시 방지

### 안정성 기준

- 셸 프로세스가 종료되어도 앱은 종료되지 않아야 한다.
- 잘못된 escape sequence를 받아도 앱은 크래시하지 않아야 한다.
- 대량 출력이 발생해도 UI가 장시간 멈추지 않아야 한다.
- 창 크기 변경, 복사/붙여넣기, 프로세스 종료 같은 기본 상호작용에서 예측 가능하게 동작해야 한다.

### 제외할 기능

- 탭
- 분할 화면
- GPU 가속 렌더링
- 플러그인 시스템
- 원격 동기화
- 테마 편집 UI
- 복잡한 설정 화면
- Windows/Linux 지원
